# SETUP_WIZARD.md — 首次使用引导

> 第一次使用小说 Agent？跟着这个向导走，15 分钟配置好你的写作环境。

---

## 开始之前

在开始之前，你需要：

1. 一个支持 Agent 的环境（OpenClaw / Claude Code / Codex 等）
2. 将此仓库 clone 到本地
3. 对着 Agent 说：「你好，我的小说 Agent，帮我初始化新书」

或者手动将 `workspace-template/` 下的所有文件复制到你的小说目录，然后逐一填写。

---

## 第一步：书名和题材

回答这几个问题：

```
书名：《              》
题材：□ 都市  □ 末世  □ 仙侠  □ 西幻  □ 悬疑  □ 校园  □ 其他：
一句话简介：
```

**示例**：
```
书名：《星河纪元》
题材：☑ 都市  □ 其他：星际科幻
一句话简介：2117年，地球成为银河系最大的黑市贸易港，主角靠捡星际垃圾为生，
          意外卷入了人类与硅基生命的终极战争。
```

---

## 第二步：选择叙事框架

你的故事适合哪种节奏？

| 框架 | 适合题材 | 特点 |
|------|---------|------|
| **三段式引擎** | 都市异能、末世、打怪升级 | 开局→发展→高潮，节奏清晰 |
| **起承转合** | 情感细腻、群像戏、文青向 | 起→承→转→合，起伏更细腻 |
| **事件驱动** | 快节奏、高潮迭起、爽文 | 以关键事件为节点，不拖沓 |

```
选择：□ 三段式引擎  □ 起承转合  □ 事件驱动
```

---

## 第三步：选择自主程度

你想让 Agent 多自主？

| 级别 | 说明 | 适合人群 |
|------|------|---------|
| L1 全自动 | Agent 全权决定，写完通知你 | 大纲已经非常完整，想快速出稿 |
| L2 半自动 | 日常写作 Agent 定，关键节点需你确认 | 有大纲，但细节想把握 |
| L3 低自动 | 核心方向你定，Agent 执行细节 | 习惯把控全局，不想失控 |
| L4 纯辅助 | 你说写什么，Agent 就写什么 | 新手，或者想完全自己决策 |

```
选择：□ L1  □ L2  □ L3  □ L4
```

## 第四步：填写世界观

参考 `workspace-template/world_template.md`，回答：

```
## 地理/世界构造
这个世界分几个区域？
每个区域的特点是什么？

## 力量体系/超凡规则（如有）
这个世界有什么特殊能力？
能力有等级划分吗？规则是什么？

## 主要势力
有哪些势力/组织/门派？
它们之间的关系是什么？（敌对/盟友/中立）

## 核心矛盾
故事的核心冲突是什么？
```

```
- 每章目标字数：   字
- 对话比例上限：   %
- 是否需要章节钩子：□ 是  □ 否
- 其他偏好：
```

---

## 第七步：生成初始大纲

完成后，对 Agent 说：

「我的书已经配置好了，帮我生成一个大纲框架」

Agent 会：
1. 读取你填写的 world.md 和 characters/
2. 根据你选择的叙事框架生成分卷大纲
3. 写入 `outline/volumes/volume_01.md`
4. 给你确认和调整

---

## 第八步：开始写作

配置完成！现在你可以：

```
「写第一章」
「帮我规划第二卷的大纲」
「新增一个配角」
「检查一下前五章有没有矛盾」
「这个角色的年龄是不是有问题」
```

## 常见问题

**Q: 我不确定我的世界观设定够不够好，怎么办？**
A: 先填一个基础版，Agent 会在写作过程中帮你发现漏洞并提问。

**Q: 之后可以改自主程度吗？**
A: 可以，随时对 Agent 说「我想把自主程度从 L2 改成 L1」。

**Q: 框架选错了能换吗？**
A: 能，对 Agent 说「我想从三段式换成起承转合」。

**Q: 没有灵感，Agent 能帮我想情节吗？**
A: 可以，Agent 会基于你的世界观和大纲给你建议，但你决定要不要用。
