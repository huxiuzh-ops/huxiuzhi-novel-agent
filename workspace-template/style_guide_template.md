# 写作风格偏好

> 告诉 Agent 你喜欢什么样的写作风格，Agent 会在写作时遵守。

---

## 基本设定

- **叙事视角**：□ 第一人称（我）  □ 第三人称全知  □ 第三人称限制（他/她）
- **目标字数**：每章 _____ 字以上（建议 3000-5000 字）
- **章节结尾**：□ 必须有钩子  □ 可以没有  □ 随意

---

## 文风偏好

| 维度 | 选项 |
|------|------|
| 句子长度 | □ 长句偏多（细腻）  □ 短句偏多（利落）  □ 混合 |
| 描写密度 | □ 重描写（氛围感强）  □ 轻描写（节奏快）  □ 混合 |
| 对话风格 | □ 生活化  □ 书面化  □ 精简有力 |
| 情绪基调 | □ 沉重压抑  □ 中性  □ 热血高燃  □ 冷幽默 |

---

## 具体偏好（可多选）

- [ ] 动作场面要有清晰的物理逻辑，不能写得太玄
- [ ] 对话不要啰嗦，每句要有信息量
- [ ] 内心独白要有，但不能整段整段
- [ ] 环境描写要精简，一两句话点到位
- [ ] 紧张场面用短句，快节奏
- [ ] 感情场面可以稍微细腻一些
- [ ] 章节结尾要有悬念，吸引人追更
- [ ] 不要太多解释性文字，让读者自己品味
- [ ] 脏话可以用，但要克制，不满天飞
- [ ] 其他：_____________________

---

## 禁止项（Agent 必须避免）

- [ ] 不写任何血腥暴力内容
- [ ] 不写任何色情内容
- [ ] 不写任何敏感政治内容
- [ ] 不写任何真实地名/真实人物
- [ ] 对话连续不超过 5 句（必须有动作/心理/场景穿插）
- [ ] 单一场景/视角不超过 2000 字要换一下节奏
- [ ] 其他禁止项：_____________________

## 角色声音示例

*如果你的某个角色有特定的说话风格，在这里写出来，Agent 会记住。*

**示例：**
- 陈末（主角）：话少、冷幽默、紧张时用调侃缓解、不说废话
  - ❌ "我很害怕，我们可能会死在这里"
  - ✅ "看来今天不是个好日子，不过至少不是最坏的那种"

- 系统（如果你的书有系统/AI角色）：毒舌、嘲讽、关键时刻给提示
  - ❌ "您的选择很危险，请三思"
  - ✅ "说真的，我都替你的智商着急。"
