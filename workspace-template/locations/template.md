# 地点模板

> 填写地点设定。每个在故事中出现过两次以上的地点都应该有档案。
> 少于两次的龙套地点可以直接写在章节里，不需要单独档案。

---

## 基本信息

```yaml
id: loc_placename       # 地点稳定ID，格式: loc_{地名拼音小写}
name:                    # 地点主名
aliases: []              # 别名，如 ["黑水渡口", "小渡口"]
file: world.md           # 通常在世界设定里，也可以是具体章节
kind: ruins              # 类型，见下表
region:                  # 所属区域
first_appearance: ch001  # 首次出场
last_appearance: ch001   # 最近出场
```

**kind 允许值**：`city` / `town` / `village` / `ruins` / `transport_hub` / `camp` / `building` / `underground` / `wilderness`
