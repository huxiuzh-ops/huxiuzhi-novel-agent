# 装备/道具模板

> 填写重要的装备或道具。普通物品不需要档案。
> 只有影响剧情、或在多处出现的道具需要填写。

---

## 基本信息

```yaml
id: item_itemname        # 道具稳定ID，格式: item_{名字拼音小写}
name:                    # 道具名称
category: weapon          # weapon / consumable / tool / intel / other
key_property:             # 关键属性（这个道具最核心的特点）
owner: char_playername    # 当前持有者ID（引用 characters.json）
```

**category 允许值**：`weapon` / `consumable` / `tool` / `intel` / `other`
