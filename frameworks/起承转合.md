# 起承转合

> 起承转合是中国传统文学的叙事框架，适合情感细腻、人物成长、群像戏、文青向作品。

---

## 核心理念

**起（铺）→ 承（推）→ 转（折）→ 合（收）**

四步循环，层层递进。节奏不如三段式快，但情绪更细腻。适合需要读者代入、共情、思考的故事。

---

## 四个步骤的含义

### 起（Introduction / 铺陈）

**目的**：建立场景、人物、情绪状态

**要点**：
- 点明时间、地点、环境
- 呈现人物当前状态
- 暗示情绪基调
- 不需要强烈冲突，但要"有意思"

**技巧**：
- 场景描写要有质感（光线/声音/气味/温度）
- 用细节建立人物性格
- 第一句话要抓住人（不要"今天是..."这种平淡开场）

**示例**：
> 雨已经下了三天。地下室的木板上渗出一层白霜，陈末把最后一块压缩饼干掰成两半，
> 小的那块推到新生面前。新生没说话，她已经三天没说话了。

---

### 承（Development / 推进）

**目的**：事件发展，矛盾积累，关系变化

**要点**：
- 有具体事件在推进
- 人物有选择和行动
- 内在情绪或外在冲突在积累
- 可以有小的波折和反复

**禁忌**：
- 承的部分写太长（拖节奏）
- 没有实质进展（"他们聊了很久"）
- 反复同一个情绪（不要写了十章还在"他很纠结"

**技巧**：
- 每隔一段要给一点"新东西"（新信息/新人物/新线索）
- 人物要有主动行为，不能只是被动反应

---

### 转（Turning Point / 转折）

**目的**：打破平衡，制造意外，改变走向

**这是最关键的步骤。**

**要点**：
- 要"意外但合理"
- 可以是外在事件（意外来信/人物死亡/计划失败）
- 也可以是内在转变（人物顿悟/决定/改变）
- 转折后故事方向要改变

**禁忌**：
- 转折太突兀（没有铺垫）
- 转折太弱（读者猜到了）
- 转折没有代价（改变了但什么都没失去）

**技巧**：
- 转的时候可以用短句，加快节奏
- 留白一点，让读者自己品味
- 转折后要有短暂的空白，让情绪沉淀

---

### 合（Resolution / 收束）

**目的**：情绪落点，承接下一章

**注意**：这里的"合"不是"完结"，而是本章的收束。

**要点**：
- 给本章一个落点（情绪落下来）
- 暗示下一步的走向
- 可以留钩子，但不要太大
- 情绪基调要稳定一下，不能戛然而止

**示例**：
> 陈末看着新生的眼睛，那里面有末世三年没有出现过的东西。
> 不是希望，是比希望更实在的东西——是理由。

---

## 起承转合检查清单

### 每章检查
- [ ] 起：200字内进入场景？
- [ ] 起：场景有质感（光/声/味/温度）？
- [ ] 承：有具体事件在推进，不是原地踏步？
- [ ] 承：有给"新东西"（新线索/新人物/新信息）？
- [ ] 转：是意外但合理的转折？
- [ ] 转：转折后故事方向有改变？
- [ ] 合：情绪有落点？
- [ ] 合：留了钩子？

### 每卷检查
- [ ] 起：5章内建立核心冲突？
- [ ] 承：单卷内有没有多个起承转合的循环（螺旋上升）？
- [ ] 转：每卷至少有一个让人意外的转折？
- [ ] 合：结尾的情绪落点够不够？

## 起承转合 vs 三段式

| 维度 | 起承转合 | 三段式 |
|------|---------|--------|
| 节奏 | 慢，更细腻 | 快，更紧凑 |
| 适合 | 情感、人物成长、群像 | 节奏、冲突、高潮 |
| 读者情绪 | 沉浸感强 | 紧张感强 |
| 单章密度 | 相对平均 | 开头结尾重 |
| 写作难度 | 容易拖，需要控制节奏 | 容易僵，需要有变化 |

---

## 适合题材

✅ 情感文、成长文、群像文、悬疑文、文学性强的作品

❌ 不适合：打怪升级型爽文、需要持续高频爽点的文

---

## 起承转合在长篇中的应用

长篇不是一次性"起承转合"完，而是**多个起承转合的串联**：

```
第一章：起 → 承 → 转 → 合
第二章：起 → 承 → 转 → 合（起要承接上章的"合"）
...
第N卷：起 → 承 → 转 → 合
第N+1卷：起（承接上卷的"合"）→ ...
```

下一章的"起"要自然承接上一章的"合"，这是串联的关键。
