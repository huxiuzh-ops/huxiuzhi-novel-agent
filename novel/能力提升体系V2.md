# 一、基础设定

## 1.1 什么是段位

段位是绑定者综合实力的核心标尺，由三条分支的等级总和决定。

计算公式：
> 综合段位分 = (战斗系等级 × 3) + (生存系等级 × 2) + (情报系等级 × 2) + 剧情点修正

段位名称从低到高依次为：

## 1.2 剧情点（CP）

剧情点仍是核心资源，但不再直接决定段位，而是作为「升级燃料」和「商店货币」。

### 1.2.1 获取途径

| 事件等级 | 剧情点范围 | 难度 | 示例 |
|---------|-----------|------|------|
| D级 | +5 ~ +10 | 简单 | 捡到物资、获取一般信息 |
| C级 | +10 ~ +20 | 中等 | 营地互帮、探索危险区 |
| B级 | +20 ~ +35 | 困难 | 击败变异体、拯救重要NPC |
| A级 | +35 ~ +50 | 极难 | 以少敌多、揭露重大阴谋 |
| S级 | +50 ~ +80 | 史诗 | 改变战争走向、击杀BOSS |
| 超S级 | +100+ | ??? | 超越当前剧情框架 |

### 1.2.2 选项修正（同样适用扣分）

| 选项类型 | 剧情点变化 | 说明 |
|---------|-----------|------|
| 勇敢抉择 | +5 ~ +15 | 主动迎接危险 |
| 揭露真相 | +10 ~ +30 | 揭露阴谋或秘密 |
| 愚蠢作死 | -20 ~ -50 | 明显找死行为 |
| 背信弃义 | -10 ~ -30 | 出卖盟友或信任 |
| 自我牺牲 | +20 ~ +50 | 为他人牺牲自己 |
| 意外发现 | +5 ~ +20 | 探索时随机触发 |
| 系统嘲讽 | -5 | 故意激怒系统（后果自负） |

### 1.2.3 CP消耗主要用途

| 消耗用途 | 费用 | 说明 |

## 1.3 每日配额系统

每日配额决定每天能「主动使用」几次技能。

| 段位 | 每日配额 | 可用蓄力槽 |
|------|---------|-----------|
| 黑铁 | 3次 | 0 |
| 青铜 | 4次 | 1 |
| 白银 | 5次 | 2 |
| 黄金 | 6次 | 2 |
| 铂金 | 7次 | 3 |
| 钻石+ | 8次 | 3 |

**配额耗尽后果：**
- 系统强制休眠24小时，期间所有技能冷却
- 休眠期间受到的攻击所进入的战斗，自动进入「系统旁观模式」
- 系统旁观模式下，系统仅提供数据分析，不会主动干预战斗

**配额恢复：**
- 每日0点自动恢复
- 剧情点强行刷新（20 CP/次），可立即恢复全部配额

# 二、三系分支系统

## 2.1 分支总览

绑定者的成长分为三条独立分支，每条有专属技能树和等级成长。

| 分支 | 核心定位 | 升级依赖 | 战斗系主要技能 |
|------|---------|---------|--------------|
| 战斗系 | 进攻、伤害、击杀 | 战斗次数+CP投入 | 暴力突破/精准打击 |
| 生存系 | 防御、恢复、隐匿 | 存活天数+CP投入 | 身体强化/隐匿潜行 |
| 情报系 | 功能、预判、解码 | 剧情事件+CP投入 | 威胁预判/系统解码 |

> **升级规则**：每升1级需要「本系经验值」+「剧情点」
> - 每升1级所需经验值 = 50 × (当前等级 + 1)
> - 每升1级所需CP = 30 + (当前等级 × 20)

# 三、能力共振系统

## 3.1 共振基础

当两条或三条分支同时达到特定等级时，可激活「共振技能」。

共振技能是超越普通技能的存在，效果强大但触发条件苛刻。

### 3.1.1 双系共振

| 组合 | 共振技能名 | 效果 | 触发条件 |
|------|-----------|------|---------|
| 战斗Lv2 + 生存Lv2 | 「战场绞肉机」 | 战斗中生命值越低，伤害越高（最低+100%） | 两系均达到Lv2 |
| 战斗Lv3 + 情报Lv3 | 「读心猎手」 | 对T2智慧体可直接攻击其思维核心，秒杀率+30% | 两系均达到Lv3 |

# 四、欠债与危机系统

## 4.1 欠债机制

欠债（负分）是系统对离谱行为的惩罚，但它也有清偿机制。

### 4.1.1 每日欠债计算

| 欠债区间 | 日欠债增量 | 系统态度 |
|---------|-----------|---------|
| 0 | 无 | 正常毒舌 |
| -1 ~ -49 | -1/天 | 「一般失望」 |
| -50 ~ -99 | -3/天 | 「相当嫌弃」 |
| -100 ~ -199 | -5/天 + 商店涨价20% | 「我已经不想说话了」 |
| -200 ~ -249 | -10/天 + 商店涨价50% + 配额-1 | 「危险区域，强行介入警戒」 |
| **-250** | **强制介入** | 系统强制接管，剧情点清零 |

### 4.1.2 欠债清偿方式

| 清偿方式 | 效果 | 限制 |
|---------|------|------|
| 剧情事件 | 正面剧情可获得「清偿点数」直接抵消 | 每次事件最多抵消30点 |
| CP转账 | CP余额可直接抵消欠债 | 每50 CP抵消10点欠债 |
| 段位下滑 | 段位下滑时自动清偿：每下滑1个小段位抵消20点 | 仅在欠债超过-100时触发 |
| 系统任务 | 完成系统发布的「惩罚任务」 | 每日最多1次，奖励10~30点清偿 |

### 4.1.3 欠债历史记录（示例）

```
第1周：欠债-20（首次作死）
第2周：欠债-60（无保护进入丧尸潮中心）
第3周：段位下滑，抵扣40点，剩余-20
第4周：欠债-10（正面剧情抵消）
第5周：欠债+15（主动正面行为）
第6周：欠债0（清零！）
```

---

## 4.2 危机触发事件

当欠债进入危险区域时，系统会主动触发「危机事件」：

| 欠债区间 | 触发事件 | 后果 |
|---------|---------|------|
| -50 | 「警告」 | 系统提示音加强，商店涨价 |
| -100 | 「强制任务」 | 必须完成一个任务，否则触发强制介入 |
| -150 | 「系统接管警告」 | 系统开始接管主角部分行动 |
| -200 | 「强行介入」 | 系统接管战斗，每次3分钟 |
| **-250** | **「强制介入」** | **系统接管主角所有行动5分钟，之后恢复控制但欠债清零** |

# 七、系统态度（扩展版）

系统态度不仅影响毒舌对白，还影响实际功能。

| 态度档位 | 触发条件 | 对话风格 | 商店折扣 | 隐藏事件概率 | 特殊权限 |
|---------|---------|---------|---------|------------|---------|
| **愉悦** | 段位+10或欠债清零 | 「本系统今天心情不错。」 | 价格-10% | +5% | 可主动询问系统意见 |
| **正常** | 段位无变化 | 「说真的，您继续这样我也习惯了。」 | 标准价 | 基准 | 无 |
| **嫌弃** | 段位下滑或欠债增加 | 「我都替您丢人。」 | 价格+10% | -3% | 无 |
| **危险** | 欠债达到-100 | 「您已经踩在红线上了。」 | 价格+25% | -10% | 商店部分商品锁定 |
| **强制介入** | 欠债达到-250 | 「够了——本作者亲自来。」 | 商店关闭 | 100%触发危机 | 系统完全接管 |

> **隐藏事件**：系统态度愉悦时，有概率触发「系统额外奖励」或「剧情分支」等隐藏内容。

> **本系统的核心理念**：您不是在「刷数值」，而是在「写角色」。
> 
> 您的每一个选择都在定义陈末这个人。
> 
> 他是暴力突破的战斗狂人？
> 还是深谋远虑的末日商人？

> 他是一个想让系统都头疼的变数？

### 给您的建议

- **不要均衡发展**：三系全修是大师阶段的事，初期必须选择一到两条主攻路线
- **欠债是战略工具**：适度的欠债可以解锁特定的「惩罚任务」，而某些惩罚任务奖励极高
- **共振技能才是质变**：双系共振是你和其他绑定者拉开差距的关键
- **系统态度决定上限**：保持愉悦态度能让你解锁更多隐藏内容
