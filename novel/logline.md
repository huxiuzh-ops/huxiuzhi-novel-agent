# Logline

## 末世 + 系统 + 作者吐槽 模式

### 模板结构
在[末世背景]中，[普通人/废物]主角意外绑定[作者型吐槽系统]，
系统代表作者本人，天天吐槽主角的决策，
主角在系统的"毒舌指导"下，一边被嘲讽一边升级打怪，最终[称霸/拯救/探索真相]。

### 本书 Logline（待填写）
> 在丧尸横行、秩序崩坏的末世中，普通上班族**陈末**意外激活了一个自称"作者本人"的毒舌系统。
> 系统讨厌无脑爽文，嫌弃主角的决定比剧情还离谱，却不得不配合他活下去。
> 主角在系统的花式吐槽中，一边升级一边探索末世真相，最终发现——

### 核心卖点
1. **吐槽系统** = 作者的替身，主角每做一个"主角行为"，系统就开骂
2. **反差爽感**：主角不是龙傲天，是普通人，但有系统辅助
3. **黑色幽默**：系统对套路的嫌弃 = 读者对套路的吐槽
4. **末世升级**：生存压力 + 资源争夺 + 势力博弈
