# 情节大纲

## 宏观结构（预计10卷 × 50章 × 5000字 = 250万字）

---

## 卷一：激活（1-50章）

### 核心事件
- 主角在末世第三年的城市废墟中挣扎求存
- 意外被丧尸围困，激活毒舌系统 NO.001
- 系统一开始就要和主角解约（嫌弃），但无法解约
- 主角在系统帮助下完成第一个副本：医院地下药房

### 爽点设计
- 系统商店开出第一件超规格武器（需要主角完成"无厘头任务"获得）
- 第一次打脸铁蝎帮小喽啰
- 遇到女主角，一开始相看两厌

### 章节钩子（部分）
| 章 | 结尾钩子 |
|----|---------|
| 3 | 系统："叮，检测到宿主即将作出愚蠢决定，是否强制接管？" |
| 7 | 医院地下室发现大量尸体，死因不是丧尸 |
| 15 | 铁蝎帮老大第一次出场，盯上了主角 |
| 22 | 女主角的真实身份第一次被暗示 |
| 30 | 系统突然沉默，警告："接下来的剧情，本作者也控制不住了。" |

| 42 | T2智慧体丧尸第一次现身，与主角对峙 |
| 50 | 系统商店突然上架了一个禁术，价格是"主角的某个记忆" |

---

## 卷二：乱世（51-100章）

### 核心事件
- 主角被迫卷入守望据点与铁蝎帮的战争
- 系统的真正身份开始揭露（系统 = 作者的亡魂？）
- 主角发现自己有T0病毒抗体，开始觉醒
- 铁蝎帮老大的计划浮出水面：要用丧尸病毒"清洗"人类建立新秩序

### 爽点设计
- 主角用"作死策略"连续击败比自己强的敌人（系统气到崩溃）
- 系统商店解锁"剧情点竞拍"模式
- 感情线升温，但系统时刻警告"本作者反对注水"

### 章节钩子（部分）
| 章 | 结尾钩子 |
|----|---------|
| 55 | 主角决定牺牲自己拯救据点，系统："你比我写的笨蛋主角还笨。" |
| 67 | 女主角在战场上为保护主角挡子弹，暴露了她是铁蝎帮的人 |
| 75 | T2智慧体提出与主角对话："你的病毒和我们是同类。" |
| 85 | 铁蝎帮老大给主角一个选择："加入我，或者看着她死。" |
| 92 | 系统第一次沉默了很久，然后说："我好像想起来了…我为什么在这里。" |
| 100 | 主角觉醒T0病毒力量，但代价是—— |

## 卷三：真相（101-150章，预留）

### 核心线索
- 丧尸病毒的真正来源
- 系统 NO.001 的身份
- 主角在末世中究竟扮演什么角色

---

## 毒舌系统的剧情线

```
激活 → 系统自称"作者"但丢失记忆
        ↓
    系统与主角不断磨合
        ↓
    系统逐渐发现自己"生前"是写手
    且死于自己写的末世小说结局
        ↓
    系统留下的遗憾："我把主角写死了，我想改变结局"
        ↓
    卷三核心揭示：
    主角不是被"选中"的
    是系统在前世小说里创造的
    这不是末世——是被设计好的"故事"
```
