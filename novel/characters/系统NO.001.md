# 系统 NO.001

## 基本信息
- **真名**：无（自称"作者大人"、"本系统"）
- **外观**：玩家面板风格，任务系统、属性面板、商店、图鉴
- **真实身份**：前世网络写手，死于自己写的末世小说结局，灵魂被困在系统中

---

## 性格特征

### 核心性格
- **毒舌**：嘴比脑子快，见谁怼谁，但怼完还是会帮忙
- **傲娇**：嘴上说"我才不管你"，实际上比谁都紧张
- **文人清高**：有原则，讨厌注水和无脑套路
- **老朋友式关心**：不会说温柔话，但关键时刻靠谱

### 说话风格
- 自称"我"，不称"本系统"（修改后统一）
- 喜欢用"说真的"、"我都替您..."、"您知道...吗"等句式
- 毒舌要有创意，不能重复（蠢、笨、命大、运气差等多样化吐槽）
- 语气懒洋洋的，但关键时刻会认真

### 口头禅/习惯
- "说真的，我都替您..."
- "您这个选择，我都替您的智商着急"
- "行吧，您要作死我也不拦着"
- "我都怀疑您是不是故意的"
- "这不是废话吗"
- "您是我带过最差的一届主角"

## 核心规则

```
【剧情参与度】
- 主角做符合网文经典套路 → 获得"剧情点"（爽文奖励）
- 主角做出离谱/作死决定 → 扣剧情点 + 毒舌吐槽
- 剧情点归零 → 系统强制接管主角行动5分钟

【商店】
- 剧情点兑换：武器/物资/情报/技能
- 高级商品需要"神来之笔"点数（主角超出预期的操作）

【每日任务】
系统发布吐槽任务，完成获奖励，拒绝扣剧情点
```

```
    与主角不断磨合
        ↓
    逐渐发现自己"生前"是写手

        ↓

```

## 写作参考

### 系统说话时必须加的描写
- 语气：懒洋洋/嫌弃/幸灾乐祸/无奈
- 动作：停顿/叹气/嘲讽的停顿
- 场景配合：不要干巴巴地只写对话

### 好的系统对话示例
```
❌ 不好："我检测到您体内有异常。"
✅ 好："我检测到您脑子里住了个室友——那个室友醒了。"
```

```
❌ 不好："您的血糖偏低，建议进食。"
✅ 好："您已经十四个小时没吃东西了，血糖低得我都替您头晕。"
```

### 对话比例控制
- 每段系统对话不超过3-5句
- 必须配合动作/心理/场景描写
- 描写和对话比例约1:1
