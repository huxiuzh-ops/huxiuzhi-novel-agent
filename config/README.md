# config/ — 框架级配置

> 这些配置文件是**项目无关的通用参数**，适用于任何小说项目。
> **不要在这里写任何具体小说内容（角色名、势力名、具体章节等）。**

---

## 文件说明

| 文件 | 用途 |
|------|------|
| `project.yaml` | 项目基础信息（标题/语言/题材/叙事框架/自主程度） |
| `writing.yaml` | 写作规则（字数/对话上限/章节钩子/描写比例） |
| `validation.yaml` | 验证规则（一致性检查/伏笔追踪/报告等级） |
| `platforms.yaml` | 多平台配置（OpenClaw/Claude/Codex/API 各平台的入口和启用状态） |

---

## 配置读取优先级

```
config/*.yaml（框架默认值）
       ↓ 覆盖
项目根目录 novel-agent.yaml（用户填写的项目配置）
       ↓ 覆盖
环境变量 NOVBL_AGENT_*（运行时覆盖）
```

---

## 在 Setup Wizard 中的作用

`SETUP_WIZARD.md` 引导用户填写的答案，会写入用户小说项目的 `novel-agent.yaml`。

`config/` 里的文件是**框架默认值**，始终保持项目无关。

---

## 扩展 config/

如果需要新增配置项（如 `llm.yaml` 存放模型供应商设置），在对应文件中添加即可。
